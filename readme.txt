== Description ==

Tweak the performance and automated settings of your Wordpress wsite by checking or unchecking the numerous options on the settings page.

Activate the plugin and select the desired settings in the 'Simple Tweak' settings page!

== Installation ==

1. Unzip the archive in the wp-content/plugins folder.
2. Activate via the plugin activation screens.
